# API Эндпоинт для получения медианной скорости выполнения заданий

## Описание
API эндпоинт для получения медианной скорости выполнения заданий по социальной сети, типу действия и количеству действий. Результат кэшируется на 24 часа для оптимизации производительности.

## URL
```
GET /api/median-speed/
```

## Параметры запроса
- `social_network` (обязательный) - код социальной сети (например: TWITTER, LINKEDIN, GITHUB)
- `action` (обязательный) - код типа действия (например: LIKE, REPOST, COMMENT, FOLLOW)
- `actions_count` (обязательный) - количество действий (положительное целое число)

## Пример запроса
```
GET /api/median-speed/?social_network=TWITTER&action=LIKE&actions_count=10
```

## Пример ответа
```json
{
    "social_network": "TWITTER",
    "action": "LIKE",
    "actions_count": 10,
    "median_speed_minutes": 45.5,
    "cached_at": "2024-01-15T10:30:00Z",
    "cache_expires_in": "24 hours"
}
```

## Логика работы
1. **Точное совпадение**: Если есть задания с точным количеством действий для указанной комбинации социальной сети и действия, вычисляется медианная скорость выполнения за вчерашний день.

2. **Fallback на социальную сеть**: Если точных совпадений нет, вычисляется медианная скорость по всем заданиям данной социальной сети и типа действия за вчерашний день.

3. **Дефолтное значение**: Если данных вообще нет, возвращается значение по умолчанию (60 минут).

## Кэширование
- Результат кэшируется на 24 часа (86400 секунд)
- Ключ кэша: `median_speed_{social_network}_{action}_{actions_count}`
- При повторном запросе с теми же параметрами возвращается закэшированный результат

## Ошибки
- `400` - Отсутствуют обязательные параметры или неверный формат
- `404` - Социальная сеть или тип действия не найдены
- `500` - Внутренняя ошибка сервера

## Примеры использования

### Получение медианной скорости для Twitter лайков (10 действий)
```
GET /api/median-speed/?social_network=TWITTER&action=LIKE&actions_count=10
```

### Получение медианной скорости для LinkedIn репостов (5 действий)
```
GET /api/median-speed/?social_network=LINKEDIN&action=REPOST&actions_count=5
```

### Получение медианной скорости для GitHub комментариев (3 действия)
```
GET /api/median-speed/?social_network=GITHUB&action=COMMENT&actions_count=3
```
