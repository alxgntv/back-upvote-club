CSV Format Requirements:

  • Required column: slug - used to identify/create landing pages
  • Optional columns: title, social_network (ID, code, or name), action, meta_title, meta_description, h1, content, page_type, page_blocks, short_description, long_description, redirect_url, faq (JSON format)
  • If a landing with the given slug exists, it will be updated; otherwise, a new one will be created
  • The CSV file should use UTF-8 encoding
  • social_network can be provided as ID (number), code (e.g., "TWITTER"), or name (e.g., "Twitter")
  • faq should be valid JSON array, e.g.: [{"q": "Question?", "a": "Answer"}]
